## Key Features Highlighted

### Innovation Documentation

1. **Ephemeral + Persistent Workspace Duality**
   - Only platform offering both modes
   - Enables stateless execution AND multi-step workflows
   - Competitive advantage vs. GitHub Copilot, Cursor, Replit

2. **Model Tier Resource Allocation**
   - Automatic right-sizing based on model capability
   - Cost optimization through intelligent resource allocation
   - Flagship (2 vCPU, 4 GB) vs Fast (0.5 vCPU, 1 GB)

3. **ULID-Based Dispatch IDs**
   - Time-ordered, sortable identifiers
   - Natural chronological queries in DynamoDB
   - Better than UUID for time-series data

4. **Cost Event Decoupling**
   - EventBridge async emission (zero latency)
   - Enables flexible pricing models
   - Future-proof for enterprise billing

5. **Context Injection System**
   - zeOS integration for project awareness
   - 3 levels: minimal (600 tokens), standard (1,200), full (1,800)
   - Session continuity across agent executions

### Performance Documentation

| Operation | Latency | Notes |
|-----------|---------|-------|
| Cold Start | 25-45s | Container provisioning |
| Warm Start | ~1.2s | Reused worker (25-30x faster) |
| Status Poll | 50-150ms | DynamoDB query |
| Fleet Status | 100-200ms | Aggregated state |
| v1 SSM Dispatch | 2-5s | Legacy |
| v2 HTTP Dispatch | 100-300ms | **10-50x faster** |

### Cost Transparency

- Infrastructure: ~$50 (ALB + NAT + Secrets)
- Agent Subscriptions: $170 (Claude $100 + Codex $20 + Gemini $50)

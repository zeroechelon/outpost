> "do other zeos project agents need their own AWS key to initiate calls via Outpost?"

**Answer:** No. Outpost v2.0 uses HTTP API with API key authentication (X-API-Key header). No AWS credentials required by client applications. Outpost control plane manages all AWS interactions (DynamoDB, ECS, S3, Secrets Manager) on behalf of tenants.
